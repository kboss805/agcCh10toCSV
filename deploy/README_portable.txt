agcCh10toCSV v2.3.0 - Portable Edition
=======================================

Run agcCh10toCSV.exe to launch the application.

Settings are stored in the settings/ folder next to the executable.
All preferences (theme, recent files, directories) are saved locally
in this folder - nothing is written to the Windows registry or %APPDATA%.
