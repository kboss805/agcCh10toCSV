agcCh10toCSV v3.1.0 - What's New
=================================

Plot PDF Export
  - Export PDF button in the plot window
  - Save current plot to high-quality PDF file
  - File save dialog with automatic .pdf extension
  - Success/error logging with clickable file link
  - Export button disabled when no data is loaded
  - Improved button spacing in axis controls

  - Auto-scale Y axis with manual override
  - Time window controls for X axis

  - Auto-assigned color palette
  - Dark/light theme support

v2.4.0 - Batch Ch10 Processing
